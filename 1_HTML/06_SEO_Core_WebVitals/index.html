Core Web Vitals:-

CLS- Cumulative Layout Shift (CLS) measures visual stability. It quantifies how much the content on a web page shifts while it is loading. A low CLS score indicates that the page is stable and elements do not move around unexpectedly, providing a better user experience.

LCP- Largest Contentful Paint (LCP) measures loading performance. It tracks the time it takes for the largest content element to appear on the screen. A fast LCP score indicates that the page loads quickly and provides a good user experience. 2.5 seconds or less is considered a good LCP score.

FID- First Input Delay (FID) measures interactivity. It quantifies the time from when a user first interacts with a page (e.g., clicks a link or presses a button) to when the browser responds to that interaction. A low FID score indicates that the page is responsive and provides a good user experience.
